Freeze Notes(26.2.0+)

How To Use

1. Open in Chart Editor.

2. Select Freeze Note In Note Type Selection.

3. Place Freeze Note Note Type In Chart Grid.

4. Select Freeze Note Event In Event Selection.

5. Place Freeze Note Event In Chart Grid.

About Event:
Value 1: Duration Freeze Note(Also Note Range to Freeze Between Current Duration to Current Duration + Duration)(Default: 1).
Value 2: Should It Destroy Note After Freeze?(0 = false, 1 = true)(Default: 1)(Info: when set not destroy(1). It will move event and Freeze Note time to value1 second more earlier. and after freeze the note can be pressable)

Warning: if use Freeze Note event while old event still run it might will broken. use with caution.

Result

1. Destroy(Value2: 1)

2. Not Destroy(Value2: 0)
