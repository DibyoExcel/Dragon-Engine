Custom Mod Settings and GLobal Settings(26.4.0+)

This will make mod has own setting. such like mod config or mod sensitive content toggle etc.

Mod Settings

Open in Mods Menu.

file location: mods/%modfoldername%/settings.json.


Json Field:

  • storageKey: storage key for setting. also can use for lua variable prefix.(default: %modfoldername%).
  • bgColor: custom setting background color.(default: #ea71fd)
  • optionsList: array list options to make

Options Field:

  • name: Options Label Name.
  • variable: Options Variable(must be unique). also can use for lua variable.
  • type: Options Type.(allowed value: int, float, percent, bool, string, stringfree(26.5.0+) and hex(26.5.0+)).
  • defaultValue: Default Value Settings.
  • options(string type only): Array list for selected text.

for more Options List Field Check in source.


example json settings.json:

{
    "bgColor": "#FF0000",
    "storageKey": "customModSetting",
    "optionsList": [
        {"name": "bool setting", "variable": "boolsetting", "type": "bool", "defaultValue": true},
        {"name": "float setting", "variable": "floatsetting", "type": "float", "defaultValue": 1.1},
        {"name": "int setting", "variable": "intsetting", "type": "int", "defaultValue": 1},
        {"name": "percent setting", "variable": "percentsetting", "type": "percent", "defaultValue": 0.9},
        {"name": "string setting", "variable": "stringsetting", "type": "string", "defaultValue": "type 1", "options": ["type 1", "type 2", "type 3"]},
        {"name": "string free setting", "variable": "stringfreesetting", "type": "stringfree", "defaultValue": "text default"},
        {"name": "hex setting", "variable": "hexsetting", "type": "hex", "defaultValue": "FF0000"}
    ]   
}

Global Mod Settings

Open in Main Options Menu.

file location: mods/settings.json.


Json Field:

  • storageKey: storage key for setting. also can use for lua variable prefix.(default: _global).
  • settingName: setting name for title and options name. also can use for lua variable prefix.(default: Global Setting).
  • bgColor: custom setting background color.(default: #ea71fd)
  • optionsList: array list options to make

Options Field:

  • name: Options Label Name.
  • variable: Options Variable(must be unique). also can use for lua variable.
  • type: Options Type.(allowed value: int, float, percent, bool, string and stringfree(26.5.0+) and hex(26.5.0+)).
  • defaultValue: Default Value Settings.
  • options(string type only): Array list for selected text.

for more Options List Field Check in source.


example json settings.json:

{
    "bgColor": "#FF0000",
    "storageKey": "customGlobalModSetting",
    "settingName": "Custom Global Mod Setting"
    "optionsList": [
        {"name": "bool setting", "variable": "boolsetting", "type": "bool", "defaultValue": true},
        {"name": "float setting", "variable": "floatsetting", "type": "float", "defaultValue": 1.1},
        {"name": "int setting", "variable": "intsetting", "type": "int", "defaultValue": 1},
        {"name": "percent setting", "variable": "percentsetting", "type": "percent", "defaultValue": 0.9},
        {"name": "string setting", "variable": "stringsetting", "type": "string", "defaultValue": "type 1", "options": ["type 1", "type 2", "type 3"]},
        {"name": "string free setting", "variable": "stringfreesetting", "type": "stringfree", "defaultValue": "text default"},
        {"name": "hex setting", "variable": "hexsetting", "type": "hex", "defaultValue": "FF0000"}
    ]   
}

Lua Usage

1. For lua usage it use storageKey_variable. example: storageKey is a coolMods. And settings variable is mySetting. In lua variable will be coolMods_mySetting.

2. If storageKey not set the default in lua variable. in Mod Setting will setting_ and in Global Mods will be globalSetting_.
