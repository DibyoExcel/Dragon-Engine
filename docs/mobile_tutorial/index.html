How to install

1. open download link from Github.

2. Scroll down until found Releases section.

3. choose version to install. example i use 1.6.3.11m1.

4. press "Dragon.Engine-Android.zip" to download zip.

5. extract the zip from download.

6. install the app just like how you install apk normally.

7. open the app. usually will promped about permission, just accepted.

8. or if not show permission promp, you can set manually by open Setting>App Management>App List>Friday Night Funkin': Dragon Engine>Permission>Storage>Allow management of all files

9. open again it should promp about ".DragonEngine" folder has been created. and will closed app.

10. back to apk file then view files.

11. open assets folder.

12. copy "assets" and "mods"(optional) to .DragonEngine folder(be sure show hidden folder turn on).

13. open the app and now it should ready to play.

How to install Mods

1. download any mod to play.

2. extract the mod zip.

3. extract to .DragonEngine>mode>.

4. then extract.

5. be sure mods folder structure like this.

6. done you just install the mods.
